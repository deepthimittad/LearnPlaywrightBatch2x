# VS Code Keyboard Shortcuts (Windows)

The most commonly used Visual Studio Code default shortcuts on Windows.

---

## General

| Shortcut | Action |
|:---|:---|
| `Ctrl + Shift + P` | Command Palette |
| `Ctrl + P` | Quick Open (Go to File) |
| `Ctrl + Shift + N` | New Window |
| `Ctrl + W` | Close Window/Editor |
| `Ctrl + ,` | Open Settings |
| `Ctrl + K Ctrl + S` | Open Keyboard Shortcuts |

## Basic Editing

| Shortcut | Action |
|:---|:---|
| `Ctrl + X` | Cut line (empty selection) |
| `Ctrl + C` | Copy line (empty selection) |
| `Ctrl + Shift + K` | Delete line |
| `Ctrl + Enter` | Insert line below |
| `Ctrl + Shift + Enter` | Insert line above |
| `Alt + Up/Down` | Move line up/down |
| `Shift + Alt + Up/Down` | Copy line up/down |
| `Ctrl + /` | Toggle line comment |
| `Shift + Alt + A` | Toggle block comment |

## Multi-cursor

| Shortcut | Action |
|:---|:---|
| `Alt + Click` | Insert cursor |
| `Ctrl + Alt + Up/Down` | Insert cursor above/below |
| `Ctrl + D` | Add selection to next find match |
| `Ctrl + Shift + L` | Select all occurrences of current selection |

## Navigation

| Shortcut | Action |
|:---|:---|
| `Ctrl + G` | Go to Line |
| `Ctrl + T` | Show all Symbols |
| `F12` | Go to Definition |
| `Alt + F12` | Peek Definition |
| `Ctrl + Shift + F10` | Peek References |
| `Ctrl + Shift + Tab` | Navigate editor group history |

## Search & Replace

| Shortcut | Action |
|:---|:---|
| `Ctrl + F` | Find |
| `Ctrl + H` | Replace |
| `Ctrl + Shift + F` | Find in Files |
| `Ctrl + Shift + H` | Replace in Files |

## IntelliSense

| Shortcut | Action |
|:---|:---|
| `Ctrl + Space` | Trigger suggestion |
| `Ctrl + Shift + Space` | Trigger parameter hints |
| `F2` | Rename Symbol |

## Editor Management

| Shortcut | Action |
|:---|:---|
| `Ctrl + N` | New File |
| `Ctrl + Shift + T` | Reopen closed editor |
| `Ctrl + Tab` | Next editor |
| `Ctrl + 1/2/3` | Focus 1st/2nd/3rd editor group |
| `Ctrl + K Ctrl + W` | Close all editors |

## Debugging

| Shortcut | Action |
|:---|:---|
| `F9` | Toggle Breakpoint |
| `F5` | Start/Continue |
| `Shift + F5` | Stop |
| `F10` | Step Over |
| `F11` | Step Into |
| `Shift + F11` | Step Out |

## Integrated Terminal

| Shortcut | Action |
|:---|:---|
| `` Ctrl + ` `` | Toggle Terminal |
| `` Ctrl + Shift + ` `` | New Terminal |
| `Ctrl + C` | Copy selection (in terminal) |
| `Ctrl + V` | Paste into terminal |

---

> **Tip:** You can view and customize all shortcuts by pressing **`Ctrl + K Ctrl + S`** (or going to **File > Preferences > Keyboard Shortcuts**).
